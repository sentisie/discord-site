Bot Commands

All available commands for Resonix bot

/play

Play a track from SoundCloud. You can use URL or just enter the track name

/play [url or track name]

/stop

Stop playback and clear the queue

/stop

/skip

Skip the current track and play the next one in queue

/skip

/loop

Enable/disable track looping. The current track will play repeatedly

/loop

/queue

Display the current queue of tracks

/queue

/shuffle

Shuffle all tracks in the current queue

/shuffle

/volume

Adjust the playback volume (0-100)

/volume [0-100]

/nowplaying

Show information about the currently playing track

/nowplaying

loop

Loops the current song

/loop

loopq

Loop the current song queue

/loopq

lyrics

Get the lyrics of a song

/lyrics

move

Moves track to a different position

/move [position]

nowplaying

Shows the song currently playing in the voice channel

/nowplaying

pause

Pauses the current playing track

/pause

ping

View the bot's latency

/ping

play

Searches and plays the requested song
Supports: SoundCloud

/play [url or track name]

previous

Go back to the previous song

/previous

queue

Shows the current queue

/queue

reload

Reload all commands

/reload

remove

Remove track you don't want from queue

/remove [position]

replay

Replay current playing track

/replay

resume

Resume current track

/resume

save

Saves current song to your DM's

/save

search

Search for a song

/search [query]

seek

Seek to a specific time in the current song

/seek [time]

shuffle

Randomizes the queue

/shuffle

skip

Skip the current song

/skip

skipto

Skip to a specific song in the queue

/skipto [position]

stats

Get information about the bot

/stats

stop

Stops whatever the bot is playing and leaves the voice channel
(This command will clear the queue)

/stop

summon

Summons the bot to the channel

/summon

volume

Change the volume of the current song

/volume [0-100]
